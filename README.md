## 🚀 Quick Revision Cheatsheet

### React Must-Know Concepts
- **Virtual DOM** → JS copy → diff → minimal real DOM update
- **useState** → use functional update `prev => prev + 1` for dependent state
- **useEffect** → `[]` = mount once, `[dep]` = on change, return = cleanup
- **useRef** → no re-render on change, DOM access
- **useCallback** → stable function reference (with React.memo)
- **useMemo** → expensive computed value
- **React.memo** → skip re-render if props unchanged (shallow compare)
- **Code Splitting** → `React.lazy(() => import('./Component'))`
- **Error Boundary** → class component, `getDerivedStateFromError`
- **Stale Closure** → use functional state update or add to deps array

### JavaScript Must-Know Concepts
- **Closure** → inner function remembers outer scope variables
- **Event Loop** → Call Stack → Web API → Microtask Queue → Callback Queue
- **Promise states** → Pending → Fulfilled / Rejected
- **async/await** → syntactic sugar over Promises
- **`this`** → depends on how function is called, not where written
- **Arrow function** → no own `this`, inherits from parent
- **Debounce** → wait for silence (search input)
- **Throttle** → limit to once per interval (scroll/resize)
- **Shallow copy** → spread `{...obj}` — nested objects still shared
- **Deep copy** → `structuredClone(obj)` or `JSON.parse(JSON.stringify(obj))`

### HTML5 Must-Know Concepts
- **Semantic HTML** → `<header>`, `<nav>`, `<main>`, `<article>`, `<footer>` → SEO + Accessibility
- **localStorage** → persists forever, 5-10MB, not sent to server
- **sessionStorage** → cleared on tab close
- **Cookies** → sent to server with every request, HttpOnly = XSS safe
- **JWT** → prefer HttpOnly cookie over localStorage (XSS protection)
- **ARIA** → `role`, `aria-label`, `aria-hidden`, `aria-expanded`

### CSS3 Must-Know Concepts
- **box-sizing: border-box** → always set globally
- **Flexbox** → 1D layout (row OR column)
- **Grid** → 2D layout (rows AND columns)
- **position: sticky** → relative until scroll threshold, then fixed
- **CSS Variables** → `--primary: #60a5fa` → dynamic, runtime changeable
- **Specificity** → Inline > ID > Class > Element
- **Animate** → `transform` + `opacity` only (GPU-accelerated, no reflow)
- **Mobile-first** → write base styles for mobile, use `min-width` queries for larger
